~/apps/Rosetta/demos/protocol_captures/2015

Applications: 
mp_dock : Membrane protein-protein docking application
mp_dock_setup : Setup application for membrane protein-protein docking application
mp_symdock : Membrane symmetric protein-protein docking application
mp_span_from_pdb : Predict trans-membrane spans from the structure
mp_transform : Transform protein into membrane coordinate frame
mp_viewer : Visualize membrane protein with PyMOL viewer
mp_relax.xml : Membrane protein refinement protocol (RosettaScripts)
mp_ddG.py : ddG prediction protocol for membranes (PyRosetta)
